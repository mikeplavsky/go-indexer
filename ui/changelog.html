CUI Changelog

2.4.0 (2014-07-21)

Features

  • __cui: Add releaseDate to the global __cui variable (d9a85afe)
  • aboutBox: change aboutBox to be a service instead of directive (425e31b0, closes #964)
  • button: add button cuiType of 'link', remove cuiPane directive (461c54e8)

  • loading: loading service to notify user when promises are being resolved (96eab458, closes #939)
  • modal: change modal from directive to service (3d5aee23, closes #963)

Breaking Changes

  • <cui-about-box> is no longer a valid directive. Please use the service cuiAboutBox instead. These changes parallel those found in the modal.

For more information, please check the aboutBox documentation.

(425e31b0)

  • cuiModal is now injectable. The element <cui-modal> does not exist. Similarly, the old cuiModalService does not exist, leaving with the directive.

Any markup used within the <cui-modal> directive can be either moved to an external html document, or added in a <script> tag as an ng-template. It can then be specified where this markup is through the configuration object passed into the cuiModal.

Please see the documentation for the extensive rewrite of the modal, and how to update your modals to use this new form.

(3d5aee23)
